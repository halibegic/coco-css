Auto width#

You can add the col-auto class to the col to have auto width row.There are col-sm-auto, col-md-auto, col-lg-auto and col-xl-auto to set auto width in different viewport sizes.

col-auto
col
col-lg-auto
col
<div class="container">
    <div class="row">
        <div class="col-auto">col-auto</div>
        <div class="col">col</div>
    </div>
    
    <div class="row">
        <div class="col-6 col-lg-auto">col-lg-auto</div>
        <div class="col">col</div>
    </div>
</div>
