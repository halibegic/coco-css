Loading#

Loading indicator is used for loading or updating.

<div class="loading"></div>

Colors#

Use color utility to set loading color.

Sizes#

Add the loading-sm or loading-lg class for small or large loading size.

<!-- small -->
<div class="loading loading-sm"></div>

<!-- large -->
<div class="loading loading-lg"></div>
