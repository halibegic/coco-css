Buttons#

Buttons include simple button styles for actions in different types and sizes.

<button class="btn">default</button>
<button class="btn btn-link">link</button>

Colors#

And you can add the btn-primary, btn-secondary, btn-success, btn-error, btn-warning, btn-dark or btn-light class for additional button colors.

<button class="btn btn-primary">primary</button>
<button class="btn btn-secondary">secondary</button>
<button class="btn btn-success">success</button>
<button class="btn btn-error">error</button>
<button class="btn btn-warning">warning</button>
<button class="btn btn-info">info</button>
<button class="btn btn-dark">dark</button>
<button class="btn btn-light">light</button>

Sizes#

Add the btn-sm or btn-lg class for small or large button size. Also, you can add the btn-block class for a full-width button.

<button class="btn btn-primary btn-sm">small</button>
<button class="btn btn-primary btn-lg">large</button>
<button class="btn btn-primary btn-block">block</button>

States#

Add the active or disabled class for active or disabled button state.

<button class="btn btn-primary active">active</button>
<button class="btn btn-primary disabled">disabled</button>
<button class="btn btn-primary disabled">disabled</button>
