Progress#

The progress indicates the progress completion of a task.

<div class="progress">
    <div class="progress-bar" style="width: 50%"></div>
</div>

Colors#

Use color utility to set progress background color.

Sizes#

Add the progresss-sm or progresss-lg class for small or large progress bar.

<div class="progress progresss-sm">
    <div class="progress-bar" style="width: 75%"></div>
</div>

<div class="progress progresss-lg">
    <div class="progress-bar" style="width: 25%"></div>
</div>
