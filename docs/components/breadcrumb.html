Breadcrumb#

Breadcrumb is used show the current page navigation.

  1. Home
  2. Some Page
  3. Some Category
  4. Current Page
<nav aria-label="breadcrumb">
    <ol class="breadcrumb">
        <li class="breadcrumb-item"><a class="breadcrumb-link" href="#">Home</a></li>
        <li class="breadcrumb-item"><a class="breadcrumb-link" href="#">Some Page</a></li>
        <li class="breadcrumb-item"><a class="breadcrumb-link" href="#">Some Category</a></li>
        <li class="breadcrumb-item"><span class="breadcrumb-link active">Current Page</span></li>
    </ol>
</nav>
