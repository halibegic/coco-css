Get Started#

There are a few different ways you can include Coco within your project.

Installation#

The fastest way to use Coco is to include CSS file from the jsDelivr CDN.

<link rel="stylesheet" href="//cdn.jsdelivr.net/npm/coco-css@latest/dist/coco.min.css">

Coco is also available under npm, just run the following command:

npm install coco-css --save

then include CSS file like this:

<link rel="stylesheet" href="node_modules/coco-css/dist/coco.min.css">

Quick start#

Here is a minimalistic starter template with Coco using the jsDelivr CDN.

<!doctype html>
<html lang="en">
    <head>
        <!-- Required meta tags -->
        <meta charset="utf-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

        <!-- Page Title -->
        <title>Coco</title>

        <!-- Coco CSS -->
        <link rel="stylesheet" href="//cdn.jsdelivr.net/npm/coco-css@latest/dist/coco.min.css">
    </head>
    <body>
        <h1>Hello, world!</h1>
    </body>
</html>

Browsers support#

Coco uses Autoprefixer to make most styles compatible with earlier browsers and Normalize.css for CSS resets. For best compatibility, these browsers are recommended:

  • Chrome Last 4
  • Microsoft Edge 12+
  • Firefox ESR
  • Safari Last 4
  • Opera Last 4
  • Internet Explorer 10+
