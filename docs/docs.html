Buttons#

Buttons include simple button styles for actions in different types and sizes.

<button class="btn">default</button>
<button class="btn btn-outline">outline</button>
<button class="btn btn-link">link</button>

Colors#

And you can add the btn-dark or btn-light class for additional button colors.

<button class="btn btn-dark">dark</button>
<button class="btn btn-light">light</button>

Sizes#

Add thebtn-sm or btn-lg class for small or large button size. Also, you can add the btn-block class for a full-width button.

<button class="btn btn-sm">small</button>
<button class="btn btn-lg">large</button>
<button class="btn btn-block">block</button>

States#

Add the active or disabled class for active or disabled button size.

<button class="btn active">active</button>
<button class="btn disabled">disabled</button>

Images#

Add the img-responsive class to <img> elements. The images will scale with the parent sizes.

<img src="..." class="img-responsive" alt="...">

Crop#

Add the img-fit-contain or img-fit-cover class to fit <img> inside the element.

<img src="..." class="img-fit-cover" alt="...">
<img src="..." class="img-fit-contain" alt="...">

Layout

Containers#

Use container-sm (540px), container-md (720px), container-lg (960px), container-xl (1140px) or container-xxl (1320px) to the container for a fixed-width container with the specific max-width.

<!-- 100% width container with max-width set to container-lg (960px) -->
<div class="container container-lg">
    <div class="row">
    ...
    </div>
</div>

Grid#

col-12 (100%)
col-9 (75%)
col-6 (50%)
col-3 (25%)
<div class="container">
    <div class="row">
        <div class="col col-6">col-6</div>
        <div class="col col-3">col-3</div>
        <div class="col col-2">col-2</div>
        <div class="col col-1">col-1</div>
    </div>
</div>

Auto width#

You can add the col-auto class to the col to have auto width row. There are col-sm-auto, col-md-auto, col-lg-auto, col-xl-auto and col-xxl-auto to set auto width in different viewport sizes.

col-auto
col
col-lg-auto
col
<div class="container">
    <div class="row">
        <div class="col col-auto">col-auto</div>
        <div class="col">col</div>
    </div>
    
    <div class="row">
        <div class="col col-6 col-lg-auto">col-lg-auto</div>
        <div class="col">col</div>
    </div>
</div>

For responsive embed, add a container with the embed-responsive class. The ratio is 16:9 by default.

        <div class="modal-footer">
            <button class="btn">Save changes</button>
            <a class="btn btn-outline" href="#close">Cancel</a>
        </div>
    </div>
</div>
